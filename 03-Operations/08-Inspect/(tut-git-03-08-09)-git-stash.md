## git stash
Show stash entries and info about a specific stash entry.

-------------------------------------------------------------------------------
### Syntax
```shell
# Legend
  [] : Optional

git stash list                  : With options used by git log
git stash show [stash-entry]    : With options used by git diff
```

-------------------------------------------------------------------------------
### Examples
```shell
git stash push
git stash list
git stash show 0
git stash pop
```
