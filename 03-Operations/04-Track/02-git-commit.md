### git commit

Saves the changes to the commit history with a short message describing the 
work done.

-------------------------------------------------------------------------------

### Syntax
```
$ git commit -m <message>

# Legend
-------------------------------------------------------------------------------
[]  : Optional
<>  : Replace
|   : OR
```

### Examples
```shell
$ git commit -m 'Hello world!'
```
