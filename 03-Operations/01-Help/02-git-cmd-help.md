## git help
Display help information about a git command.

-------------------------------------------------------------------------------
### Syntax
```
$ git <command> --help  # Extended info
$ git <command> -h      # Short info

# Legend
[]  : Optional
<>  : Replace
|   : OR
```

-------------------------------------------------------------------------------
### Examples
```shell
$ git init --help   # Show info page
$ git init -h       # Show short info in the terminal
```
